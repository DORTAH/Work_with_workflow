# Задание 
## Первая часть:
1. Создать свой репозиторий (по примеру).
2. Добавит файл main.py c каким-нибудь выводом консольным
3. Добавит .github/workflows/ci.yml (скопировать мой workflow).
4. Сделать Commit и push
GitHub автоматически запустит Actions после пуша.
GitHub сам создаёт виртуальный сервер, копирует ваш код, запускает команды и сообщает — всё ли прошло успешно.
5. В отчет показать workflow и результат исполнения. Название Workflow должен содержать вашу фамилию 
Вторая часть:
Вы можете взять мою мини программу на Python  с тестом (скидывал выше в чат), или реализовать свою на удобном для вас языке.
1. Добавьте к базовому workflow, который сделали раньше:
- a. Блок с установкой всех необходимых доп. Инстурментов (линтер, форматтер, тестер)
- b. блок запуска линтера
- c. блок запуска форматтера
- d. блок запуска теста
2. Попробуйте специально видоизменить (сломать код), а также сделать неправильное оформление мини программки, сделать пуш, и проверить результат.
3. почините код и запустите
Отчет это сркины основных шагов, код, результат как на скрине# Work_with_workflow
